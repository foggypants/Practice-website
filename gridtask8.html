What problem does auto-fit solve?

It automatically creates as many columns as can fit in the row based on the available space, removing the need to manually set column counts.

Why is minmax important?

It sets a safety limit by defining a minimum width (e.g., 200px) so items don't get too small, while allowing them to grow to 1fr to fill extra space.

Do items shrink too much?

No, because the 'min' value in minmax prevents them from shrinking below your specified limit; instead, they wrap to the next row.

Is this layout mobile friendly?

Yes, it is perfectly mobile-friendly because the columns automatically stack vertically once the screen width becomes narrower than the minimum item width.

Why are media queries not needed here?

The browser does all the calculation and wrapping automatically based on the minmax rules, making the layout responsive by default.
